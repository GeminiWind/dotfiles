# Dotfiles

Personal configuration file to setup new environment in UNIX system

```shell
.
├── bootstrap
├── p10k // configuration files for p10k
├── vim // configuration files for vim
│   └── config
│       ├── closetag.vim
│       ├── coc.vim
│       ├── floaterm.vim
│       ├── fzf.vim
│       └── nerdtree.vim
└── zsh // configuration files for zsh
```
